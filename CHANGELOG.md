# Changelog

## 4.2.0

- 移除豆瓣搜索、详情获取和封面下载对 Electron 浏览器环境的依赖，恢复桌面端、移动端、浏览器端和 Docker 环境支持。
- 新增移动端全屏读书笔记工作台，并针对导航、搜索结果、书籍详情和设置弹窗进行移动交互适配。
- 修复移动端工作台及插件业务弹窗无法纵向滑动、输入框与设置项高度异常的问题。
- 修复微信读书外部头像等 HTTP 图片在 HTTPS 客户端中的混合内容问题。
- 导入书籍时增加规范化书名去重，避免缺少 ISBN 或 Book ID 时生成同名读书笔记失败。
- 新增 `npm run release` 自动发布流程，统一完成版本更新、依赖校验、构建、打包、提交、Tag 和 GitHub Release。
