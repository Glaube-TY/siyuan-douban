<h2 style="margin:0 0 .7em;color:#6b3f1d;">✨ What Can It Do for You?</h2>
<p style="margin:0;color:#5b4a35;">If you manage books, organize excerpts, and write reading notes in SiYuan, this plugin can help you chain together repetitive workflows like "searching books, creating databases, generating documents, and syncing notes."</p>
</div>
<div style="margin-top:1.3em;display:grid;grid-template-columns:repeat(2,minmax(0,1fr));gap:14px;">
<div style="padding:1.15em;background:#fffaf0;border:1px solid #dfc998;border-radius:14px;box-shadow:0 5px 16px rgba(91,58,30,.06);">
<h3 style="margin:0 0 .45em;color:#7a3f18;">🔍 Quick Book Search</h3>
<p style="margin:0;color:#665640;">Search by ISBN or title keywords to quickly fetch book metadata.</p>
</div>
<div style="padding:1.15em;background:#fffaf0;border:1px solid #dfc998;border-radius:14px;box-shadow:0 5px 16px rgba(91,58,30,.06);">
<h3 style="margin:0 0 .45em;color:#7a3f18;">📥 Auto Database Entry</h3>
<p style="margin:0;color:#665640;">Automatically fill title, author, publisher, ISBN, cover, and other info into your database.</p>
</div>
<div style="padding:1.15em;background:#fffaf0;border:1px solid #dfc998;border-radius:14px;box-shadow:0 5px 16px rgba(91,58,30,.06);">
<h3 style="margin:0 0 .45em;color:#7a3f18;">📝 Note Templates</h3>
<p style="margin:0;color:#665640;">Generate reading notes based on customizable templates with consistent structure.</p>
</div>
<div style="padding:1.15em;background:#fffaf0;border:1px solid #dfc998;border-radius:14px;box-shadow:0 5px 16px rgba(91,58,30,.06);">
<h3 style="margin:0 0 .45em;color:#7a3f18;">📗 WeRead Sync</h3>
<p style="margin:0;color:#665640;">Sync highlights, thoughts, book reviews, popular highlights, public account articles, and reading statistics via API Key.</p>
</div>
</div>
<div style="margin-top:1.4em;padding:1.45em;background:#fbf3dc;border-left:5px solid #9a3412;border-radius:14px;box-shadow:0 6px 18px rgba(122,63,24,.06);">
<h2 style="margin:0 0 .8em;color:#8a2e12;">🚀 New WeRead Sync</h2>
<ul style="margin:0;padding-left:1.25em;color:#5b4a35;">
<li>Uses API Key / official API mode, no longer relying on the old login-state sync method.</li>
<li>Supports regular books, public account accounts, and public account article sync.</li>
<li>Supports books with notes, new book confirmation, bookshelf view, and local document matching.</li>
<li>Supports reading statistics dashboard to view weekly, monthly, yearly, and total reading data.</li>
</ul>
</div>
